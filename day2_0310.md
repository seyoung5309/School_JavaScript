# 자바스크립트란

동적인 웹 사이트 제작을 위해 사용되는 것.
1990년대 넷스케이프의 브랜던 마이크가 mocha라는 라이브스크립트를 개발하여 최종적으로 자바스크립트가 되었다.

유럽 컴퓨터 제조 연합에서 스크립트 표준을 만들었다. 모던 스크립트라는 ES6버전을 만들었다. 

바닐라 자바스크립트 -> 어떤 플랫폼에서든 작동하는, 그치만 어렵다? 

---

### 프론트 엔드 개발자 과정 
_html -> css -> 반응형웹(Bootstrap) -> JavaScript -> React, Vue_

# 자바스크립트의 삽입
1. <head> (3)
2. <body> (2)
3. 외부 파일로 삽입 (1)
4. 인라인(inline) _(비추천)_ (4)
